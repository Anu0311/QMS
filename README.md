1. Vendor Management System (QMS)

Title:
Vendor Management System | ASP.NET Core MVC | SQL Server

Description:

Developed a comprehensive Vendor Management System to streamline vendor onboarding, certification tracking, compliance monitoring, and inspection management.

Key Features:

Vendor registration and profile management
Certification and document tracking
Third-party inspection management
Reporting dashboards
Workflow automation
File upload and document management

Technology Stack:
ASP.NET Core MVC, C#, SQL Server, Entity Framework Core, jQuery, AJAX, Bootstrap

Outcome:
Improved visibility into vendor compliance and reduced manual tracking through centralized workflows.
